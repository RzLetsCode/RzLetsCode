## ⚡ The Philosophy: Architecture Over Syntax

Most platforms teach you how to call an API. **Companies hire you to build systems.** **code2career_ai** is a systematic framework designed to move builders beyond "Tutorial Hell." We focus on the high-leverage skills that define senior AI roles: **Architecture Design**, **LLMOps**, **Agentic Workflows**, and **System Evaluation**.

---

## 🎯 Targeted Path Matrix
*Choose your track and bypass the chaotic "Tutorial Loop."*

> ### 🎓 Students & Freshers
> **Focus:** Foundations to Proof-of-Work.
> * **Outcome:** Start AI from scratch with a clear, industry-vetted sequence. Turn basic scripts into **High-Signal Assets** that prove you are ready for internships and entry-level roles.

> ### 🔄 Career Switchers
> **Focus:** Gap Analysis & Strategic Pivot.
> * **Outcome:** Identify exactly where your previous background meets AI. Reposition your experience into a **Strategic Narrative** that convinces hiring managers of your cross-functional value.

> ### 🏗️ Builders & Aspirants
> **Focus:** Production Refinement.
> * **Outcome:** Move beyond "it works on my machine" to "it works in production." Refine architecture clarity, evaluation pipelines, and documentation—making your GitHub **Recruiter-Ready.**

## 🧭 The 6-Phase Success Framework
*A goal-oriented journey from confusion to a career-ready narrative.*

1.  **Understand the Path** | Identify exactly how AI components (LLMs, Vector DBs, Agents) connect.
2.  **Learn with Real Systems** | Study simplified versions of production-style architectures.
3.  **Build Meaningful Projects** | Create engineering solutions you can defend with confidence during interviews.
4.  **Choose Tools with Purpose** | Master tools that match real job descriptions (Azure, Pinecone, LangGraph).
5.  **Tell the Right Story** | Explain your decisions in a way that resonates with recruiters and lead engineers.
6.  **Improve through Feedback** | Refine your projects through honest, professional audits and code reviews.

## 🛠️ The Project Curriculum
*Engineering problems solved through code. This curriculum moves learners from basic API interactions to autonomous, enterprise-grade deployments.*

| Phase | Project Identity | The Engineering Challenge | Tech Stack |
| :--- | :--- | :--- | :--- |
| **01** | **Contextual Memory Agent** | Managing stateless LLM API calls with persistent conversation history. | `LangChain`, `OpenAI` |
| **02** | **YouTube Insight Extractor** | Bypassing token limits using Map-Reduce summarization patterns. | `Python`, `Claude 3.5` |
| **03** | **Semantic Resume Matcher** | Overcoming keyword limitations in HR tech using dense vector embeddings. | `Pinecone`, `SBERT` |
| **04** | **Insurance Claim Assistant** | Mitigating hallucinations and implementing strict PII masking guardrails. | `Azure AI`, `Presidio` |
| **05** | **GNews MCP Server** | Standardizing tool access for agents via Model Context Protocol. | `MCP`, `FastAPI` |
| **06** | **Multi-Agent Research Team** | Orchestrating autonomous AI personas for complex asynchronous tasks. | `CrewAI`, `LangGraph` |
